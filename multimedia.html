In the video above I added a <video> tag, muted (muted attribute) it and set it to autoplay (autoplay attribute) in the opening <video> tag.
I also included a picture (poster attribute) that would show while the video is loading.
I also added a the play/pause buttons and other controls (controls attribute). I set the height to 400 and the width to auto so it adjusts automatically.
In the <source> tag inside the <video> tag I added the src="" attribute and the type="" attribute which I set to type="video/mp4".
