Using the h1 header tag makes this header the most important of them all

This was written in a <p> or paragraph tag. This means that whatever is written within this tag will be displayed in a paragaph block.

Here I used a separate <p> tag in order to demonstrate to which position and style (font, font-size, padding, margin etc.) this tag defaults.

Side note: if you forgot how to encode html entities (so that the website shows symbold such as < and does not think you are trying to open a new tag), go to this page. Here you can see the charset entity name to replace each special character with.

Here you can see a <h2> type header, the second most important of them all

As you can see above, while the number behind <h> increases (e.g. from <h1> to <h2>) its importance and therefore font size decreases.

When using the paragraph tag, you can use in-line tags such as <span> tags to highlight words. In order to do that, type "style="the-style-you-want"" within the opening <span> tag.
