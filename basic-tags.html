The <em>, <i>, <strong>, <b>, and <code> tag

All the above listed tags are in-line tags. In contrast to block elements (such as the <em> <p> and <blockquote> tags), in-line elemenst do not start a new line, and take only as much space as their value. In-line tags are used within another tag such as this paragraph tag.

By putting the <em> tag around text, it is emphasized.

The <i> tag is used anytime you want to italicize text.

The <strong> tag emphasizes text by displaying it as bold.

The <b> tag on the other hand simply displays the text as bold.

Hold on, the <em> and <i> tags look exactly the same?? And so do the <strong> and <b> tags? What's the difference?
The difference between them is the emphasis! This might not be important for the majority of us, but for the program and for people with special need such as people with visual impariment the difference is monumental. Both the <i> and the <b> tag change the text purely for the purpose of style. The tags <em> and <strong> on the other hand change the style of the text as well as putting an emphasis on it. This makes a difference for e.g. programs that read the text out to people, and emphasize the sounds of the text within the <em> and <strong> tags . Then the person listening - without having to see the text - knows which part of the text to give importance to.

The <code> element shows that an inline code element has been added. If you want to add code in the same way as it is displayed in your html file, use a <pre> tag to wrap the <code> tag inside. Lists and more Lists:

The unordered list (<ul>):

  • In order to display an unordered list (one with just dots in front), you have to first create a <ul> tag
  • This <ul> tag requires an opening and a closing tag
  • Within that tag, you should wrap each item in a <li> tag.
  • each content of a single <ul>tag will be displayed as a separate list item
    • You can also make lists inside lists by nesting an unordered list inside another unordered list
    • How? you may ask
    • You simply have to create another <ul> tag within your first <ul> tag
    • Then, again, you wrap each item in a different <li> tag
  • And voil√† you have made a list inside a lits!
