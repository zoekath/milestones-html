The <p> tag, nesting:

Within this snippet we can see that the <p>/paragraph tag was used. The <p> tag requires an opening (<p>) and closing (</p>) tag. That is because in between those two tags, one can insert values such as text, numbers, other elements and other tags. The value within the <p> tags is then displayed as a paragraph block just like the text you are reading now. This is also called "nesting". Nesting proposes the idea that each element of code (such as the <p> tag) are boxes that hold content. In this case the content is what is being written right now. At the same time, as part of the content of those boxes, one can place another box such as the <a> tag in our snippet above. Here we basically nested a box inside another box. We could say that the <a> tag is the child of the <p> tag. This child box can hold content istself, such as the link to the Google homepage above.

The <a> tag, attributes:

Inside the <p> tags of the snippet, you can see that textual values ("Want to search the web? Try"), and other elements/tags (<a> tag) were added. The <a>/hyperlink tag creates a link that leads to another webpage. In the opening <a> tag, one can add certain attributes which will change the value of the tag accordingly. The attributes added to the snippet <a> tag are: href="",id="", class="", and target="". The value of the href="" attribute is: https://google.com. In our <a> tag, the href="" attribute holds the URL to the webpage that you want your page to connect to - in our case the url to the Google home page. Wihout this attribute, your page will not be linked to another website.
The class attribute and id attribute are closely related. Both attributes specifically classify elements in our code - here the link. The value of the class="" attribute is: banana. The class attribute determines the class name (here: banana) of the element. Within a CSS style sheet, we can then target this element specifically and change its layout to our liking. One page can have multiple elements under the same class. One element can also have mutliple classes. We could therefore classify this paragraph under the class="banana" and under the class="cherry" and thus style it either the same way as the snippet class (banana) or as the "cherry" class.
The value of the id="" attribute is: the-best-link. The id attribute is more specific than the class attribute and thus overrides the class styling if an element has both a class and an id attribute. In our case, the styling of id="the-best-link" would override the styling of class="banana". One element can only have one id, and the same id can only be used on one element within a page.
The value of the target="" attribute is:_blank. This attribute determines the manner in which the link will be openend. Here, the value "_blank" determines that the link shall be openend in a new tab/window.
